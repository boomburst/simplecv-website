Interacting with the Display

This isn't going to take much more work than the Hello World example. We are only introducing one new concept: the Display object.

Note: You can type 'help Display' in the SimpleCV shell for help with the Display object.

In the Hello World example, we used the show() function to display the image. The show() function is a short hand way of displaying an image, but it doesn't give you the control you can have if you use the Display object instead. For example, let's say we want our program to end when the left mouse button is clicked. The code would look like:

from SimpleCV import Camera, Display
cam = Camera()
disp = Display()

while(disp.isNotDone()):
        img = cam.getImage()
        if disp.mouseLeft:
                break
        img.save(disp)

The following line creates the display object (called disp in this example):

disp = Display()

After we create the display object, we call the function isNotDone() on it as part of the while statement. The isNotDone() checks to see if the display window is still open. If it is open, it loops through the while block to grab an image from the camera, and then display the new image. In the while block, there is the line:

if disp.mouseLeft:

If the left mouse button has been clicked, the mouseLeft() function returns true. If it's true, then the break statement is executed, and the program breaks out of the while loop. If the left mouse button hasn't been clicked, then the following line is executed instead:

        img.save(disp)

This line takes the image that was taken with the getImage() function, and saves it to the display object - or, in other words, shows it on the screen. This is the equivalent to the img.show() line in the Hello World exmaple. Using the actual display object, though, lets you check whether or not any events happened with the display, which gives you greater control.

The code for this example is in the file named display.py in the example zip file.

Hello World

The following is the SimpleCV version of the Hello World program. All it does is take a picture with the webcam attached to the computer, and then shows it on the screen.

Here is all the code that's needed to do this:

from SimpleCV import Camera 
cam = Camera()

while(1):
     img = cam.getImage()
     img.show()

Let's step through the code line by line. The first line:

from SimpleCV import Camera

imports the SimpleCV Camera library into the namespace. What does that mean? It means that you can now access all of the functions in the Camera library. If you wanted to work with functions in both the Camera and Image libraries, this line would change to:

from SimpleCV import Camera, Image

You can also import all of the SimpleCV libraries at once by using the star notation:

from SimpleCV import *

The star notation is a simple shortcut to save you from having to write out each library or class name. However, it's usually a better coding practice to specify which libraries you are importing. It makes it easier for other developers to understand what is being used by the program, and it helps remind you what you were doing if you have to revisit the code sometime in the future.

The next line in the code is:

cam = Camera()

This line creates a Camera object which lets you interact with the webcam attached to your computer.

The next few lines are part of a while loop:

while(1):
     img = cam.getImage()
     img.show()

This code will loop as long as the condition being tested by the while statement (the code inside of the parantheses after the while) remains true. In this case, the condition is the 1 in while(1). Since 1 is always true, this means the while loop will never end.

The two indented lines after the while statement are the lines that are executed as part of the loop. The indentation with these lines matters because that's how Python knows they're part of the while loop block of code. The first line:

     img = cam.getImage()

calls the getImage() function on the Camera object from line 2. This has the webcam snap an image, and save it as img. The next line: 

     img.show()

takes the image that was snapped by the webcam, and shows it to you on your computer's monitor or display. Both the show() and getImage() functions are part of the functions defined in the Camera class. This is why you needed to import it on line 1.

Since the while loop will never end, this block of code will effectively show you a running feed of the images from your webcam. To stop the program, simply close the window displaying the webcam images.

Hello World purists might object that the actual words 'Hello World' aren't actually involved in this example. It is possible to have text display on the screen, but we will leave this for a future example. If you want to play with this code yourself, the file is named helloworld.py in the example zip file.

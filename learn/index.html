Learning More About SimpleCV

This section includes example code and tutorials to help you start coding your own computer vision applications using the SimpleCV framework. There are also links to our YouTube demonstration videos where you can see SimpleCV in action. If you need help installing SimpleCV, the installation guide can help. If you would like to know more about SimpleCV itself, there is the About section below – or you can read the first chapter of our book online, which is an introduction to computer vision in general, and SimpleCV in particular. About:

SimpleCV is an open source framework — meaning that it is a collection of libraries and software that you can use to develop vision applications. It lets you work with the images or video streams that come from webcams, Kinects, FireWire and IP cameras, or mobile phones. It’s helps you build software to make your various technologies not only see the world, but understand it too.

SimpleCV is free to use, and because it’s open source, you can also modify the code if you choose to. It’s written in Python, and runs on Mac, Windows, and Ubuntu Linux. It’s developed by the engineers at Sight Machine, and it’s licensed under the BSD license.

Downloading SimpleCV

Before you click the download button below, please read the following. What you are downloading and installing is a packaged up version of SimpleCV. It is meant to try and include all the required libraries you should need to run SimpleCV. Please note that not all webcams work with SimpleCV. When you click the link it should start downloading the superpack for whatever operating system you are currently on.

Latest Stable Release: SimpleCV Version 1.2 Superpack

Once you have everything installed, you can start learning how to use SimpleCV by going to: http://examples.simplecv.org, if you are new to computer vision or SimpleCV it is recommended you purchase the book to get up to speed quickly, although it is not requried to use SimpleCV. You can also always refer to the documentation for details on what each function in SimpleCV does.

Want to run the latest and greatest?

The packaged version may not have some of the newest features maybe showcased in videos or examples posted online. If you would like to run the latest development version please follow the instructions included in the README file which is found in the download itself or is hosted in the repository here.

Need Help?

Then head over to the forums at: http://help.simplecv.org
